-Thêm thông tin danh bạ thông qua class ContactRepository, class ContactRepository khi làm sẽ làm ở background thread

-nút FloatingActionButton(fab) khi nhấn sẽ chuyển sang màn hình nhập thông tin cho danh bạ sau đó sẽ quay lại màn hình chính và
hiển thị tất cả dữ liệu trong danh bạ
